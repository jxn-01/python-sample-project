# Python Data Processing Project

A simple Python project that loads numeric values from a CSV file and prints summary statistics.

## Features
- Reads one numeric value per row from a CSV file
- Calculates count, sum, minimum, maximum, and average
- Includes unit tests and a Git-friendly project layout

## How to Run

```bash
python main.py path/to/data.csv
```

### Example CSV

Create a file called `data.csv` with numeric values, one per line:

```csv
10
20
30
45
```

Then run:

```bash
python main.py data.csv
```

## Sample Data and Demo

A sample CSV file is included at `data/sample_data.csv`.

Run the demo script with:

```bash
python demo.py
```

## Testing

Run the unit tests with:

```bash
python -m unittest
```
