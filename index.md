## Quick Start

1. Create a virtual environment (optional):

```bash
python -m venv .venv
```

2. Activate the environment and run the project:

Windows:
```bash
.venv\Scripts\activate
python main.py
```

Unix/macOS:
```bash
source .venv/bin/activate
python main.py
```
